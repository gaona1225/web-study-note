时间线（timeline）

版本：

概述：

图片轮播，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery

调用：

(引用相关资源文件,构建示例中的结构即可)

参数：

简单时间线显示区域的宽，纯数字 无单位

简单时间线显示区域的高，纯数字 无单位

时间轴时间点生成方式 "auto" 自动生成
此时根据navStartDate和navDateOffset来生成 "hand" 手动配置 此时使用navDate

自定义显示的时间点 时间数组 格式 "2012.9.1"

定义开始结束日期 生成时序时间点

游标 表示开始时间后加多少天做时间轴点 >=0的整数 总的时间点为 navDateOffset+1

时间轴对应的静态数据信息 如果配置了content则优先于url使用

ajax获取时间轴信息的链接 返回对应数目的信息
返回值为形如{"2012.9.19":"手工数据2012.9.19"}的json数据,
其中键key为对应的事件点 对应的键值为具体的html字符串

初始时激活的时间点

合并后图片路径的所在目录

方法：

暂无

销毁时间线控件方法

示例-1：

手动配置时间+手动配置内容 

                <input type="button" id="destroyBtn" value="button" />
                <div style="left:290px;top:0;" id="timeline" class="timeline"></div>
                <script>
                    $("#timeline").timeline({
                        navDate:["2012.8.30", "2012.8.31", "2012.9.1", "2012.9.2"],
                        content:["2012.8.30-(1)", "2012.8.31(2)", "2012.9.1(3)", "2012.9.2(4)"],
                        basePath:"../WidgetLib/uiBase/skins/default"
                    });
                </script>
                <script language="javascript">
					$('#destroyBtn').click(function(){
						$("#timeline").timeline('destroy') ;
					}) ;
				</script>

<div style="left:290px;top:10px;" id="timeline2" class="timeline"></div>

	var navData = ["2012.9.16", "2012.9.17", "2012.9.18", "2012.9.19"];
