高级搜索框（advancedSearch）

版本：

概述：

高级搜索框插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

advancedSearch();

参数：

advancedSearch()插件在使用时需要配置如下参数：

（字符） 搜索框的宽度

（函数） 点击搜索按钮触发的操作

（函数） 焦点进入搜索框触发的操作

方法：

暂无

示例：

高级搜索框

search all
search info
search knowledge
search disk
search xunhu
