二维码(qrcode)

版本：

概述：

二维码插件封装，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

qrcode.js, jQuery, jQuery ui

调用：

(引用相关资源文件,构建示例中的结构即可)

参数：

（字符）参数：渲染html结构生成方式 可选值为 "table", "canvas"

（数字）参数：二维码占据区域的宽

（数字）参数：二维码占据区域的高

（数字）参数：值为 < 1时 将自动计算 由qrcode.js提供 计算模块数用

（颜色值）参数：二维码区域的背景色

（颜色值）参数：二维码颜色

方法：

暂无

<script type="text/javascript">
  $('#qrcode').delegate('td>:button', 'click', function () {
      var self = $(this);
      var $tds = self.parents('tr').find('>td');
      var text = $.trim($tds.eq(0).find('>:text').val());
      var $container, render;
      if ('table' === self.val()) {
          $container = $tds.eq(2);
          render = 'table';
      } else if ('canvas' === self.val()) {
          $container = $tds.eq(3);
          render = 'canvas';
      }
      $container.empty();

      $container.qrcode({
          text: text,
          render: render
      });
  });
  // 清除
  $('#clearRender').click(function () {
      $('#qrcode').find('>tbody>tr').find('>td:last, >td:eq(2)').empty();
  });
</script>
