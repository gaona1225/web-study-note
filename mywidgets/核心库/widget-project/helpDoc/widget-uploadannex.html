上传附件（uploadannex）

当前版本：

概述：

上传附件功能，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

uploadannex();

插件在使用时需要配置如下参数：

（数字）参数：180，上传文件列表区域高度

（数组类型）参数：['jpg','gif','png','jpeg','doc','docx','xls','xlsx','txt','html','rar','zip','asp','ppt','pptx','avi','exe','swf','fla','js','xml','htm','jsp','mp3','mp4','css']：，可上传文件格式(如果设置有值表示除设置的几种文件格式其他在备选数组中的文件格式均不可上传)

（数组类型）参数：[]：不可上传文件格式(如果设置有值表示除设置的几种文件格式其他在备选数组中的文件格式均可上传)

（数组类型）参数：[]：，备选的文件格式

（函数类型）参数：{}：，通过浏览按钮上传自定义方法

（函数类型）参数：{}：，预留定义函数

（函数类型）参数：{}：，删除附件自定义方法

（函数类型）参数：{}：，dragEnter时操作

（函数类型）参数：{}：，dragOver时操作

（函数类型）参数：{}：，dragDrop时操作

（函数类型）参数：{}：，dragLeave时操作

方法：

（方法）清除上传附件

示例：

上传附件功能

浏览并选择文件，或者将文件拖放到该区域来上传文件。

            	<script>
                    $(function(){
						$('.upload-files').uploadannex();						
						$('.btn').click(function(){
							var uploadlist = $('.upload-files-list') ;
							var len = $('.upload-files-list').length ;
							for(var i=0; i<len ; i++){
								uploadlist.eq(i).attr('id',i) ;
							}
							setProgress('0',80,false) ;
						}) ;
					}) ;
			   </script>
			   <div class="upload-wrap">
                    <p class="upload-content">浏览并选择文件，或者将文件拖放到该区域来上传文件。</p>
                    <p class="upload-input"><input type="file" /></p>
                    <div class="upload-files"></div>
                    <hr class="upload-hr"/>
                    <div class="upload-btn"><input class="buttonActive" type="button" value="确定" data-width="standard"/></div>
                </div>
                <input type="button" class="btn" value="设置进度">
