标准进度条插件（progressBar）

版本：

概述：

标准进度条插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

progressBar();

参数：

progressBar()插件在使用时需要配置如下参数：

（字符） 进度条的宽度

DOM元素属性：

（字符） 参数："progressBar"

（字符） 百分比数字，不包含"%"

方法：

用于销毁该插件，使输入框恢复为原生的状态。

示例：

标准的简洁搜索框，更新数据后需要重新调用progressBar()来刷新显示效果

%（0-100）
            <script>
				$(function(){
				$(".progressBar").progressBar({width:"300px"});
					$("#set").click(function(){
						var val = $("#setVal").val();
						$("#bar").val(val);
						$(".progressBar").progressBar();
					});
				});
            </script>            
			<input id="setVal" name="" type="text" /><input id="set" type="button" class="buttonActive" value="设置" />
            <input id="bar" name="" type="text" value="30" class="progressBar" />
