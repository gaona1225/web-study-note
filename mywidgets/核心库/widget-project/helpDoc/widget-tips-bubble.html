气泡提示（bubble）

版本：

概述：

气泡提示插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

bubble();

参数：

气泡坐标，气泡x轴坐标

气泡坐标，气泡y轴坐标

气泡要显示的内容

气泡尾部的位置，可选"leftTop","leftBottom","rightTop","rightBottom"

方法：

暂无

示例：

气泡提示

 

 

            <script>
				$(function(){
						var offset = $(".help-demo").offset();
						$(document).bubble("aa",true,{
						x:offset.left+10,
						y:offset.top+4,
						content:"这里是需要显示的信息！",
						hand:"leftTop"
						});
				});
			</script>
