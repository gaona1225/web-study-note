星级评定（ratystar）

当前版本：

概述：

星级评定功能，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

ratystar();

插件在使用时需要配置如下参数：

（数字）参数：5，显示最多星星个数

（数字）参数：2：，每个星星表示的分值

（数字）参数：2：，默认显示星级,这个需要和每个星星表示的分值相除，所以建议设置值和每个星星分值成整数倍。

（布尔类型）参数：false：，是否禁用手动设置星级

方法：

（方法）禁用设置星级

（方法）启用设置星级

（方法）获取星级评定结果

（方法）清除星级评定

示例：

星级评定功能

            	<script>
                    $(function(){
						$('.exampleObj').ratystar();
						$('#disablebtn').click(function(){
							$('.exampleObj').ratystar('disable') ;
						}) ;
						$('#enablebtn').click(function(){
							$('.exampleObj').ratystar('enable') ;
						}) ;
						$('#getbtn').click(function(){
							$('#showvalue').val($('.exampleObj').ratystar('getStar')) ;
						}) ;
					}) ;
			   </script>
			   <div class="exampleObj"></div>
               <input type="button" id="getbtn" value="获取星级" />
               <label>星级评定结果</label><input type="text" id="showvalue" />
               <input type="button" id="disablebtn" value="禁用星级设置" />
               <input type="button" id="enablebtn" value="启用星级设置" />
