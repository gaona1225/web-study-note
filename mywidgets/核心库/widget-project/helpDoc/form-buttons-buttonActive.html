推荐按钮（buttonActive）

当前版本：

概述：

推荐按钮插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

buttonActive();

参数：

（字符）参数："button"

（字符）参数："buttonActive"

（字符） 标签元素中的扩展属性，允许自定义按钮的宽度。如果没有设定此属性，宽度将会自适应内容。可选参数："standard"、"数字"

方法：

使按钮转换为禁用状态。

使按钮转换为可用状态。

隐藏按钮。

显示按钮。

用于销毁该插件，使按钮恢复为原生的状态。

示例：

带有图标的标准宽度推荐按钮，可控制按钮的状态。

禁用 启用 隐藏 显示
            <script>
                    $(function(){
                        $('#disable').click(function(){
                            $('#btnDemo').buttonActive('disable');
                        });
                        $('#enable').click(function(){
                            $('#btnDemo').buttonActive('enable');
                        });
                        $('#hide').click(function(){
                            $('#btnDemo').buttonActive('hide');
                        });
                        $('#show').click(function(){
                            $('#btnDemo').buttonActive('show');
                        });
                    });
                </script>
                <input id="btnDemo" class="buttonActive" type="button" value="按钮" data-width="standard"/>
                <a href="###" id="disable">禁用</a> 
                <a href="###" id="enable">启用</a>
                <a href="###" id="hide">隐藏</a> 
                <a href="###" id="show">显示</a>
