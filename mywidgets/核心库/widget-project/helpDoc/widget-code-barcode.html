条码(barcode)

版本：

概述：

二维码插件封装，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jquery-barcode-2.0.1.js, jQuery

调用：

(引用相关资源文件,构建示例中的结构即可)

参数：

（字符）参数：条码竖杠的的宽

（数字）参数：条码竖杠的的高

（数字）参数：largeur / hauteur d'un module

（布尔）参数：是否显示编码的数字 true表示显示

（颜色值）参数：背景色

（颜色值）参数：条码颜色

（颜色值）参数：显示的编码数字的字体

（字符）参数：输出方式 可选值 css（只有该项支持ie）, svg, bmp, canvas

（数字）参数：canvas输出时在画布中的x坐标

（数字）参数：canvas输出时在画布中的y坐标

方法：

暂无

示例：

<script type="text/javascript">
    var $codeNum = $('#codeNum');
    var $type = $('#type'); // select
    var $barWidth = $('#barWidth');
    var $barHeight = $('#barHeight'); 
    var $moduleSize = $('#moduleSize'); 
    var $showHRI = $('#showHRI'); // select
    var $bgColor = $('#bgColor');
    var $color = $('#color');
    var $fontSize = $('#fontSize');
    var $posX = $('#posX');
    var $posY = $('#posY');
    var $output = $('#output'); // select

    var $result = $('#result');
    var option = {};
    $('#generate').click(function () {
        // 清空已有
        $result.empty();
        if ($type.val() === '-1') {
            alert('请选择编码类型');
            $type.focus();
            return;
        }
        if ($showHRI.val() === '-1') {
            alert('请选择是否显示编码的数字');
            $showHRI.focus();
            return;   
        }
        // 获取最新参数
        option = {
            barWidth: $barWidth.val(),
            barHeight: $barHeight.val(),
            moduleSize: $moduleSize.val(),
            showHRI: $showHRI.val() === 'true',
            bgColor: $bgColor.val(),
            color: $color.val(),
            fontSize: $fontSize.val(),
            posX: $posX.val(),
            posY: $posY.val(),
            output: $output.val()
        };
        $result.barcode(
            $codeNum.val(),
            $type.val(),
            option
        );
    });
</script>
