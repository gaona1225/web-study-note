图片懒加载效果（lazyImage）

当前版本：

概述：

图片懒加载效果，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

$('selector').lazyImage({....});

插件在使用时需要配置如下参数：

（布尔型类型）参数：true 表示只处理当前上下文下的符合条件的img; false 则根据属性值来获取需要懒加载的图片 默认 true

（布尔型类型）参数：true 初始化时插件自动绑定加载事件事件 false 手动调用load方法进行

（字符串类型）参数：限定标识要懒加载处理的图片 默认 data-lazyImg

（字符串类型）参数：懒加载图片的占位图片路径

方法：

无

示例：

懒加载功能(1,自动绑定;2,手动绑定)

            	<script>
                $(function(){
                  function _init(ctxSelector, isHungry) {
                      var baseUrl = '.';
                      var img = [
                          '/images/lazyImage/1.jpg',
                          '/images/lazyImage/2.png'
                      ];
                      var i, len;
                      var domHtml = '';
                      for (i = 0, len = img.length; i < len; i++) {
                          domHtml += '<div class="test_item">' +
                                          '<img data-original="' + baseUrl + img[i] + '" data-lazyImg="true" data-x="' +
                                          ctxSelector + i + '"/>' +
                                      '</div>';
                      }
                      $(ctxSelector).append(domHtml).lazyImage({
                          placeHolder: 'images/lazyImage/blank.gif',
                          // 是否自动绑定事件
                          isHungry: isHungry
                      });
                  }
                  _init('#imgCon', true);
                  _init('#imgCon2', false);
                  $('#triggerLoad').bind('click', function() {
                    // 手动绑定事件
                    $('#imgCon2').lazyImage('load');
                  });
                }) ;
              </script>
              <div id="imgCon"></div>  
              <div id="imgCon2">
                  <input type="button" id="triggerLoad" value="手动绑定加载"/>
              </div>
