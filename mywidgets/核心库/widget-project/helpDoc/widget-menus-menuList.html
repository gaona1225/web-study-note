标准菜单列表（menuList）

版本：

概述：

标准菜单列表，针对版本1.2+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

menuList();

参数：

菜单数据的URL，或者是json格式的字符串

弹出菜单的id

数据回显标志的boolean值，默认为false，数据不回显。

字数限制，默认为5。

动作函数

方法：

用于销毁该插件。

示例-1：

标准菜单列表

链接菜单▼ 
            <a id="li" href="javaScript:void(0)">链接菜单▼</a>
             <script>
             $(function(){
				 $("#li").menuList({json:"../WidgetLib/source/menuList/json2.json",listId:"bb",dateShow:true,numControl:6});
				});
             </script>

示例-2：

按钮菜单列表

            <input class="buttonPro menuList" id="hi" type="button" value="选项" data-img="../WidgetLib/uiBase/skins/base/icon_system/16px/menu.png" data-type="menu" />
             <script>
             $(function(){
			 	 var jsonData = '[{"id":"data1","menuList":"data1"},{"id":"data2","menuList":"data2"},{"id":"data3","menuList":"data3"}]';
				 $("#hi").menuList({
					json:jsonData,
					listId:"aa",
					fn:function(){
						$("#aa").live("click",function(event){
							if($(event.target).is("li")){
								alert("你点击ID是: "+$(event.target).attr("id"));
							}
						});
					}
				});
			});
             </script>
