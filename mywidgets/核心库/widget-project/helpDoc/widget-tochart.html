表格转图表（tochart）

当前版本：

概述：

文本裁剪控件，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

tochart();

插件在使用时需要配置如下参数：

（字符串类型）参数：'bar'，转换的图表形式，可选值有：'bar'-柱状图,'area'-区域图,'pie'-饼状图,'line'-线性图

（数字类型）参数：null，转换的图表宽度,当未手动设置宽度或设置值为null,取表格的宽度。

（数字类型）参数：null，转换的图表高度,当未手动设置高度或设置值为null,取表格的高度。

（布尔类型）参数：true，转换的图表是否显示表格的标题

（字符串类型）参数：null，转换的图表显示的标题

（字符串类型）参数：'*'，转换的图表行过滤条件设置

（字符串类型）参数：'*'，转换的图表列过滤条件设置

（数组类型）参数：['#be1e2d','#666699','#92d5ea','#ee8310','#8d10ee','#5a3b16','#26a4ed','#f45a90','#e9e744']，转换的图表中显示是颜色块

（数组类型）参数：[]，转换的饼图中提示数据文本颜色，只有饼图有。

（字符串类型）参数：'x'，解析表格数据的方向，可选参数为'x','y'，被解析的字段必须放到th标签里面。

（数字类型）参数：20，转换的饼图的外边距，只有饼状图有。

（字符串类型）参数：'inside'，转换的饼图的提示数据在饼图的位置，可选参数'inside','outside',只有饼图有。

（数字类型）参数：4，转换的线性图和区域图的线的宽度，只有线性图和区域图有。

（数字类型）参数：10，转换的柱状图中每组数据之间的间距，只有柱状图有。

（数字类型）参数：1，转换的柱状图中每组数据内柱子之间的间距，只有柱状图有。

（数字类型）参数：30，转换的图表参考坐标的间距

（函数类型）参数：{}：，自定义函数

方法：

（方法）表格转图表控件

                    $(function(){
						$('.tochart').eq(0).tochart({}) ;
                        $('.tochart').eq(1).tochart({
                            type : 'area' 
                        }) ;
                        $('.tochart').eq(2).tochart({
                            type : 'line' ,
                            rowFilter: ':not(:last)',
							colFilter: ':not(:last-child)'
                        }) ;
                        $('.tochart').eq(3).tochart({
                            type : 'pie' , 
                            pieMargin : 10 , 
                            title : '2009 Total Sales by Individual'
                        }) ;
                        $('#btn3').click(function(){
                            $('.tochart').eq(0).tochart({type:'pie'}) ;
                        }) ;
                        $('#btn1').click(function(){
                            $('.tochart').eq(1).tochart({type:'line'}) ;
                        }) ;
                        $('#btn2').click(function(){
                            $('.tochart').eq(2).tochart('destroy') ;
                        }) ;
					}) ;
			   </script>
