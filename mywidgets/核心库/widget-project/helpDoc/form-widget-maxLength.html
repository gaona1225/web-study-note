标准菜单列表（maxLength）

版本：

概述：

标准菜单列表，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

maxLength();

参数：

字数最大限制，默认为10

当超出最大值时，是否截取，默认为false

是否显示提示信息，默认为true

提示信息的class名 ，默认为"ui-maxLength-statusClass"

提示信息里面text值，默认为"您可以输入的字符个数为："

当超出最大值时，文本框的样式class名，默认为"ui-maxLength-notification"

当超出最大值时，是否显示alert提示信息，默认为false

当超出最大值时，alert提示信息所提示的内容 ，默认为"您输入的字数太多了！"

提示信息是否获得焦点时显示，失去焦点后隐藏，默认为false

事件绑定，默认为‘keyup’事件

方法：

暂无

示例-1：

input

            <input type="text" name="input_1" id="input_1"/>
             <script>
             $(function(){
				 $('#input_1').maxLength({ maxCharacters:5});
				});
             </script>

示例-2：

textarea

            <textarea cols="50" rows="5" name="textarea_2" id="textarea_2"></textarea>
             <script>
             $(function(){
				 $('#textarea_2').maxLength({
					maxCharacters:15,
					intercept:true,
					status:true,
					statusClass:"aaa",
					statusText:"不要输入超过了以下字数：",
					notificationClass:"noClass",
					showAlert:true,
					alertText:'少写点字！',
					slider:true,
					events : ['keyup']
				});
			 });
             </script>

示例-3：

设置contentEditable='true'后，div,table,p,span,body等等可编辑元素(以div和span为例)

12
span 
            <div name="div_3" id="div_3" contentEditable="true" style="height:50px;width:200px;border:1px solid #ccc">12</div>
            <span name="span_4" id="span_4" contentEditable="true" style="display:block;height:50px;width:200px;border:1px solid #ccc">12</span>
             <script>
             $(function(){
				 $('#div_3').maxLength({ maxCharacters:5});
				 $('#span_4').maxLength({ maxCharacters:5});
			 });
             </script>
