手机平台判定（isMobile）

版本：

概述：

判断是否是手机平台访问,返回布尔值，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

version.js

调用：

ui.fn.isMobile();

快捷调用：

ui.isMobile()

参数：

暂无

方法：

暂无

示例：

手机平台访问和非手机平台访问将返回不同的信息。

            <script>
				$(function(){
					if(ui.isMobile()){
						$('#show-msg').html('来访的是手机') ;
					}else{
						$('#show-msg').html('来访的不是手机') ;
					}
				}) ;
            </script>
            <span id="show-msg"></span>
