滑块组件（widgetSlider）

版本：

概述：

滑块组件插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

widgetSlider();

参数：

（字符）参数："widgetSlider"

（字符）滑块密度设置

（字符）滑块的步进值

方法：

无

示例：

滑块组件

			<script>
				$(function(){
					$(".widgetSlider").widgetSlider({width:"400px"});
				});
			</script>
			<input id="th" type="text" disabled="disabled" class="widget-textinput" />
			<input type="text" class="widgetSlider" value="10" data-size="40"/>
