图片轮播（slide）

版本：

概述：

图片轮播，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery

调用：

(引用相关资源文件,构建示例中的结构即可)

参数：

单个图片的宽，纯数字 无单位， 控制显示区的宽

单个图片的高，纯数字 无单位， 控制显示区的高

是否显示类似于1、2、3、4的控制入口 布尔型， 默认为true

控制入口容器的class值， 默认值pagination

控制入口容器中标示当前展示图片的class值，默认值current

过渡动画持续时间

默认开始显示的图片序号

动画效果;1,fade,淡入淡出;2,推入推出:h 水平;v 垂直;可选值: "h"; "v"; "slide";3,百叶窗 "shutter"

effect参数配置为"shutter"时扇叶朝向 可选值 "h" "w"

effect参数配置为"shutter"时生成的扇叶数

交互响应需要是数据存放的节点属性名，默认值data-slideParamName

构建时收集的每帧图片响应时事件响应实现

自动播放周期(设定为大于0的数字将开启自动播放)

控制手动播放时停止播放. 定为大于0的数字将开启停止播放功能,数值为停止的间隙

方法：

暂无

示例-1：

图片连续播放(淡入淡出) 

<div id="slides2">
  
  <div class="slide-container" style="overflow: hidden;position: relative;display: block;">
    <div class="slide-controller">
      <a href="http://www.baidu.com" target="_blank">
        <img src="../WidgetLib/source/slide/images/1.png"/>
      </a>
      <a href="javascript:;" style="display: block;z-index: 0;">
        <img src="../WidgetLib/source/slide/images/2.png"/>
      </a>
      <a href="javascript:;" style="display: block;z-index: 0;">
        <img src="../WidgetLib/source/slide/images/3.png"/>
      </a>
    </div>
  </div>
</div>
<script>
$(function () {
    $("#slides").slide({
        effect:"fade",
        paginationType:"hover",
        width:840,
        height:305,
        play:3000, //开启自动播放
        pause:1000,
        eventHandler:function(param){
            alert("eventHandler:" + param );
        }
    });
});
</script>
