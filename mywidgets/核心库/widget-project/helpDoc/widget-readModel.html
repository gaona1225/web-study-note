阅读模式（readModel）

当前版本：

作者：

概述：

阅读模式插件，类似视频网站的关灯效果，突出页面中的指定区域， 并读取页面中指定的区域内容以弹出层的方式显示，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

$(".ui-readModel").readModel();

注：目标div中不能有基于jqueryUI的插件，在调用hide方法时，jqueryUI会将调用所有widget插件的destroy方法

插件参数：

（数字/字符串）无默认参数，实际显示的宽度

（数字/字符串）无默认参数，实际显示的高度

（字符串）默认参数“default”，显示在窗口的中间位置，参数：'default'（原来的位置）；'center'（窗口中间位置）

（布尔值）默认参数false，是否跨iframe显示，注：最外层是否为frameset，若为frameset则没有body元素，不能实现跨iframe显示功能。

方法：

用于开灯效果，使目标对象显示为突显状态。

用于关灯效果，使目标对象恢复为原生状态。

示例-1：

默认阅读模式，及其开灯效果、关灯效果调用

readModel-阅读模式，类似视频网站的关灯效果 1、突出页面中的指定区域， 2、读取页面中指定的区域内容以弹出层的方式显示。 

            	<script>
                   $(function(){
						$('#btn1').click(function(e) {
							$('#targetId').readModel('show');
						});
					}) ;	
			   </script>
			   <input type='button' value='开灯' id='btn1' class='buttonActive'/>
               <div class='ui-readModel' id='targetId' style='background-color:#ccc;'>
					<p>
						readModel-阅读模式，类似视频网站的关灯效果
						1、突出页面中的指定区域，
						2、读取页面中指定的区域内容以弹出层的方式显示。
					</p>
				</div>

示例-2：

js重新定义图片显示，js参数覆盖默认定义效果

readModel-阅读模式，类似视频网站的关灯效果 1、突出页面中的指定区域， 2、读取页面中指定的区域内容以弹出层的方式显示。 

				<script>
					$(function(){
						$('#divID').readModel({
							height:400,
							width:400,
							model:'center',
							iframe:true	
						});
						$('#btn2').click(function(e) {
							$('#divID').readModel('show');
						});
					});
				</script>
				<input type='button' value='开灯' id='btn2' class='buttonActive'/>
			 	<div class='ui-readModel' id='divID' style='width:200px;height:200px;background-color:#ccc;'>
					<p>
						readModel-阅读模式，类似视频网站的关灯效果
						1、突出页面中的指定区域，
						2、读取页面中指定的区域内容以弹出层的方式显示。
					</p>
				</div>
