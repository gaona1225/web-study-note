单行文本框（inputText）

版本：

概述：

单行文本框插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera
v1.3版本：加入css3样式圆角和边框阴影、修改禁用启用方法

依赖：

jQuery; jQuery-UI;

调用：

inputText();

参数：

（字符）参数："text"或者"password"

方法：

使输入框转换为禁用状态。

使输入框转换为可用状态。

隐藏输入框。

显示输入框。

示例：

标准的单行文本框

禁用 启用 隐藏 显示 
				 <script>
                    $(function(){
                        $('#disable').click(function(){
                            $('#textfield').inputText('disable');
                        });
                        $('#enable').click(function(){
                            $('#textfield').inputText('enable');
                        });
                        $('#hide').click(function(){
                            $('#textfield').inputText('hide');
                        });
                        $('#show').click(function(){
                            $('#textfield').inputText('show');
                        });
                    });
                </script>
				<input type="text" name="textfield" id="textfield"  placeholder="输入你想输入的内容">
				<a href="###" id="disable">禁用</a> 
                <a href="###" id="enable">启用</a>
                <a href="###" id="hide">隐藏</a> 
                <a href="###" id="show">显示</a>
