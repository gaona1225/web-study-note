取色功能（colorpicker）

当前版本：

概述：

取色功能，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

colorpicker();

插件在使用时需要配置如下参数：

方法：

（方法）获取颜色值

（方法）销毁取色插件

示例：

取色控件

            	<script>
                    $(function(){
						$('#color').colorpicker();
						$('#btn').click(function(){
							$('#tagText').val($('#color').colorpicker('getVal')) ;
						}) ;
					}) ;
			   </script>
			   <input type="text" id="color" name="color" value="#123456" />
               <input type="button" id="btn" value="取值" />
               <input type="text" id="tagText" />
