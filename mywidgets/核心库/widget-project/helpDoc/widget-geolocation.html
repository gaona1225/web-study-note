HTML获取地理位置信息（geolocation）

当前版本：

概述：

基于HTML5获取地理位置信息，针对版本1.2+
兼容浏览器：IE9+/Chrome/Firefox/Safari/Opera等支持HTML5高版本浏览器

依赖：

jQuery; jQuery-UI;api.map.baidu.com/api?v=1.3(可选)

调用：

geolocation();

插件在使用时需要配置如下参数：

（字符）参数：'showGeo'，存放地理位置信息元素class名

（字符）参数：'您当前的浏览器不支持Geolocation,请升级浏览器'，设置不支持该属性输出的提示信息

（布尔值）参数：true,是否显示纬度

（布尔值）参数：true,是否显示经度

（布尔值）参数：true,是否显示准确度

（布尔值）参数：true,是否显示海拔高度

（布尔值）参数：true,是否显示海拔高度的精确度

（布尔值）参数：true,是否显示行进方向

（布尔值）参数：true,是否显示地面速度

方法：

（方法）获得地理位置信息-详细地址

（方法）获得地理位置信息-纬度

（方法）获得地理位置信息-经度

（方法）获得地理位置信息-准确度

（方法）获得地理位置信息-海拔高度

（方法）获得地理位置信息-海拔高度的精确度

（方法）获得地理位置信息-行进方向

（方法）获得地理位置信息-地面速度

（方法）清除地理位置信息

示例：

基于HTML5获取地理位置信息

纬度： 经度： 详细地址： 
            	<script>
                    $(function(){
						$('body').geolocation() ;
                        $('#getBtn').click(function(){
                            $('#getxBtn').val($('body').geolocation('getX')) ;
                            $('#getyBtn').val($('body').geolocation('getY')) ;
                            $('#getAddBtn').val($('body').geolocation('getAddress')) ;
                        }) ;
					}) ;
			   </script>
			   <input type="button" value="获取地理位置信息" id="getBtn"/>
                纬度：<input type="text" id="getxBtn" />
                经度：<input type="text" id="getyBtn" />
                详细地址：<input type="text" id="getAddBtn" />
