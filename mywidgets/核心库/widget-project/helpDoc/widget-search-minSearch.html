简洁搜索框（minSearch）

版本：

概述：

简洁搜索框插件，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

jQuery; jQuery-UI;

调用：

minSearch();

参数：

minSearch()插件在使用时需要配置如下参数：

（字符） 搜索框的宽度

（函数） 点击搜索按钮触发的操作

（函数） 焦点进入搜索框触发的操作

（函数） 焦点离开搜索框触发的操作

方法：

（方法）参数：将目标destory

示例：

标准的简洁搜索框

            <script>
			$(function(){ 
			$(".minSearch").minSearch({
					width:"200px",
					btnClick:function(){},
					onFocus:function(){},
					onBlur:function(){}
					});
            $('#testBtn').click(function(){
				$('#testminSearch').minSearch('destory') ;
			}) ;
			});
			</script>
            <input name="input" type="text" class="minSearch"  />
            <input name="input" type="text" class="minSearch" />
			<input name="input" type="text" class="minSearch" id="testminSearch" />
			<input name="input" class="buttonActive" type="button" value="aaa" id="testBtn"/>
