随机整数（randomInt）

版本：

概述：

生成一个既定区间内的随机整数，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

version.js

调用：

ui.fn.randomInt(min,max);

参数：

最小值

最大值

方法：

暂无

示例：

生成1到100的随机整数

            <input name="" type="text" id="rndInt" />
            <input name="" type="button" value="随机(1-100)" onclick="getRnd();" />
            <script>
			function getRnd(){
				var info = ui.fn.randomInt(1,100);
				document.getElementById('rndInt').value = info;
			}	
			</script>
