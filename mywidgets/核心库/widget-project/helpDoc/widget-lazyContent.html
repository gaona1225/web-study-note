内容懒加载(lazyContetn)

当前版本：

概述：

内容懒加载效果，针对版本1.3+
兼容浏览器：IE7+/Chrome/Firefox等浏览器

依赖：

jQuery; jQuery-UI;

调用：

$('selector').lazyContent({...});

插件在使用时需要配置如下参数：

（字符串类型）设定内容格式: '1' 表示多个平行区 类似于工作台面板及内容; '0' 多个导航 1个显示区容器

（字符串型）选择器或者对应jQuery对象, 待加载所需参数存放的上下文

（字符串类型）存放返回数据的容器元素的结构

（数字类型）css 选择器 存放请求数据的所需参数的元素

（数字类型）延迟加载的毫秒数 默认 350

（字符串类型）加载中图标的路径

（字符串类型）触发加载内容的事件 默认 click

方法：

无

示例：

1个显示区+多个导航

              <script>
                $(function() {
                  $('#navCon2').lazyContent({
                    mode: '0',
                    el: '#con2',
                    dataElem: ':button',
                    loading: 'images/lazyContent/loading.gif'
                  });
                });
              </script>
              <div class="btn-group btn-group-vertical" id="navCon2">
                <input type="button" class="btn" 
                    value="li#0" data-url="data.txt"
                    data-param="text"/>
                <input type="button" class="btn" 
                    value="li#1" data-url="data.txt"
                    data-param="text"/>
                <input type="button" class="btn" 
                    value="li#2" data-url="data.txt"
                    data-param="text"/>
                <input type="button" class="btn" 
                    value="li#3" data-url="data.txt"
                    data-param="text"/>        
              </div>
              <div class="nav_content" id="con2">

示例：

多个平行ajax块

  • li#0
  • li#1
  • li#2
  • li#3
              <script>
                $(function() {
                  //测试加载多个平行待加载项目
                  $('#nav2>li').lazyContent({
                    el: '.item_con',
                    timeout: 200,
                    event: 'customeName',
                    loading: 'images/lazyContent/loading.gif'
                  });
                  var psedoTimer = 0;
                  $('#loadAjax2').bind('click',function() {
                    var $targetLi = $('#nav2>li');
                    var len = $targetLi.length;
                    if (psedoTimer < 12 && psedoTimer >= 0) {
                      $targetLi.eq(psedoTimer++).trigger('customeName');
                    } else {
                      psedoTimer = 0;
                    }
                });
              </script>
              <input type="button" value="去加载" id="loadAjax2" class="btn"/>
              <ul class="lazy_content" id="nav2">
                  <li data-url="data/lazyContent/data1.txt"
                      data-param="text">
                      <div class="item_con">li#0</div>
                  </li>
                  <li data-url="data/lazyContent/data2.txt"
                      data-param="text">
                      <div class="item_con">li#1</div>
                  </li>
                  <li data-url="data/lazyContent/data3.txt"
                      data-param="text">

                  </li>
