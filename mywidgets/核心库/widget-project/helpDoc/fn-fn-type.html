浏览器类别-数据类别-类型判断(fn.type）

版本：

概述：

提供本地存储操作的简单封装，针对版本1.0+
兼容浏览器：IE7+/Chrome/Firefox/Safari/Opera

依赖：

version.js

调用：

是否IE浏览器; ui.fn.isIE()

是否IE7浏览器; ui.fn.isIE("7")

是否IE8浏览器; ui.fn.isIE("8")

是否IE9浏览器; ui.fn.isIE("9")

是否谷歌浏览器; ui.fn.isChrome()

是否苹果浏览器; ui.fn.isSafari()

是否火狐浏览器; ui.fn.isFx()

是否未定义; ui.fn.isUndefined()

是否Null; ui.fn.isNull()

是否函数; ui.fn.isFn( obj )

是否由非空字符组成; ui.fn.isNotEmpty( obj )

是否数字; ui.fn.isNum( obj )

是否整数; ui.fn.isInteger( obj )

是否浮点数; ui.fn.isDecimal( obj )

是否ip; ui.fn.isIp( obj )

是否链接; ui.fn.isUrl( obj )

是否合法年月日格式; ui.fn.isDateTime( obj )

是否邮箱地址; ui.fn.isEmail( obj )

是否电话号码; ui.fn.isTel( obj )

是否手机号码; ui.fn.isPhone( obj )

是否邮政编码; ui.fn.isZipcode( obj )

是否身份证号码; ui.fn.isIdCard( obj )

参数：

暂无

方法：

暂无
