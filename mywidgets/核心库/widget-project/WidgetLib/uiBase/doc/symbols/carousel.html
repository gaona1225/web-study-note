Field Summary
Field Attributes Field Name and Description
 
altBox
当鼠标悬浮在某个图片上时，显示img元素的alt属性值的标签元素
 
autoRotate
设置自动旋转方向，设置为'left'，则自动向左转动。设置为'right'，则自动向右转动，设置为'no'（默认），则不自动转动。
 
autoRotateDelay
自动转动的间隔时间
 
bringToFront
是否启用点击某张图片，该图片转动到图片正面
 
buttonLeft
向左旋转按钮对象(如果需要按钮控制，必须指定到相应的目标对象,否则失去点击按钮触发旋转功能)
 
buttonRight
向右旋转按钮对象(如果需要按钮控制，必须指定到相应的目标对象,否则失去点击按钮触发旋转功能)
 
imgClick
点击滚动图片时触发自定义点击事件
 
imgHei
旋转图片的高度（每个旋转图片的高度一致）
 
imgWid
旋转图片的宽度（每个旋转图片的宽度一致）
 
minScale
应用到最远项的最小刻度，正面项的刻度是1
 
mouseWheel
是否启动滚轮操作控制旋转
 
reflGap
图片和倒影之间的垂直间隔
 
reflHeight
设置图片的倒影高度
 
reflOpacity
指定倒影的透明度
 
speed
调整转动速度，设置为0不转动，设置为1则没有转动效果，立即切换到最终效果。
 
titleBox
当鼠标悬浮在某个图片上时，显示img元素的title属性值的标签元素
 
wrapHei
旋转图片展示区域外层高度
 
wrapWid
旋转图片展示区域外层宽度(设置宽度需要同时兼顾图片大小、目标相对于容器中心的环绕圆周位置)
 
xPos
相对于容器中心的环绕圆周水平位置
 
xRadius
水平图片环绕的半径
 
yPos
相对于容器中心的环绕圆周垂直位置
 
yRadius
垂直图片环绕的半径,你可以修改这个值来调整环绕的倾斜程度。
